# cpusim
A 8-bit CPU Simulation with Logisim

## Opcode
1.  ADD - 00001 / 10001 
2.  SUB - 00010 / 10010
3.  JMP - 00011 / 10011
4.  AND - 00100 / 10100
5.  OR  - 00101 / 10101
6.  NOT - 00110 / 10110
7.  XOR - 00111 / 10111
8.  MOV - 01000 / 11000
9.  LOD - 01001 / 11001
10. STR - 01010 / 11010  
11. HLT - 11111
